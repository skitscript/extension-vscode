# Skitscript VS Code Extension (NodeJS) [![Continuous Integration](https://github.com/skitscript/extension-vscode/workflows/Continuous%20Integration/badge.svg)](https://github.com/skitscript/extension-vscode/actions) [![License](https://img.shields.io/github/license/skitscript/extension-vscode.svg)](https://github.com/skitscript/extension-vscode/blob/master/license) [![Renovate enabled](https://img.shields.io/badge/renovate-enabled-brightgreen.svg)](https://renovatebot.com/)

Adds support for Skitscript documents to Visual Studio Code.

## Installation

You may be prompted to install this extension on opening a file with an
extension of `.skitscript`.
